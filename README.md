# UDS-delphi-OMSI
Universal Door Script for OMSI 2

API:
  Variables:
    door_all_closed
    door_all_closeTarget
    doorSpeedSoundOpen_X
    doorSpeedSoundClose_X
    
   Sound Triggers:
    ev_doorlockX_on
    ev_doorlockX_off
 
 Where X is the door number from 0 to 5
 There are 3 pairs of doors (0-5)
